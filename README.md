Raidplaner-Widget
=================

* Uses only apihub.php and JSON.
* Uses Raidplaner Locales, already support german, english, french, polish and czech.
* Uses Raidplaners stylesheets (to be compatible with every supported games).
* Easy to install (1 file, 150 lines).
* Compatible with every games supported by Raidplaner.

FR:
* Utilise uniquement apihub.php et des requêtes JSON.
* Utilise les traductions de Raidplaner, supporte déjà l'Allemand, l'Anglais, le Français, le Polonais et le Tchèque.
* Utilise le style de Raidplaner (pour être compatible avec tout les jeux supportés).
* Facile à installer (1 fichier de 150 lignes).
* Compatible avec tous les jeux supportés par Raidplaner.

### How to Install ?
You just have to put a single php file in the Raidplaner root folder, edit the url, the API Token, the game name and you're ready to go !

FR:
Il vous suffit de placer un seul fichier PHP dans le dossier racine de RaidPlaner et d'y éditer l'url, le Token pour utiliser l'API et le nom du jeu, c'est tout !
